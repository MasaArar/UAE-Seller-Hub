# ProfitCalc UAE · Amazon Profit Calculator

> A free, browser-based profit calculator built for UAE-based Amazon sellers.  
> Calculate net profit, margin %, and ROI instantly — before investing in any product.

---

## What it does

Enter your product cost, selling price, shipping, and Amazon fee — and get a live breakdown of:

- **Net Profit** (AED)
- **Profit Margin** (%)
- **ROI** (%)
- Full cost breakdown (product cost, shipping, Amazon fees, VAT)

All calculations run entirely in the browser. No data is collected or stored.
